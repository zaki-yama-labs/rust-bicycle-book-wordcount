`bicycle_book_wordcount` はシンプルな文字、単語、行の出現頻度の計数機能を提供します。
CLIからは単語数の出現頻度が使えます。

```console
$ cargo run text.txt
{"bb": 1, "aa": 2, "cc": 1}
```
